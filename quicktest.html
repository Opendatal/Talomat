Konfigurationstest - Mat-O-Wahl

Dieser Test liest alle wichtigen Einstellungen aus und liefert so einen schnellen Überblick, ob man vielleicht einen Fehler in der Konfiguration gemacht hat.

Fehlerbericht

Wenn diese Meldung auch nach mehreren Minuten nicht verschwindet und JavaScript aktiviert ist, haben Sie entweder einen sehr langsamen Rechner, einen sehr umfangreichen Fragebogen oder ein echtes Problem in den Definitions-Dateien. z.B. eine Einstellung, die nicht sauber mit "Anführungszeichen" eingeschlossen ist oder eine fehlende Einstellung. Zum Beispiel:

Falsch: Fehlendes Anführungszeichen am Anfang oder Ende.
var fileQuestions = Fragen.csv";
var fileQuestions = "Fragen.csv;

Falsch: Eine vollkommen fehlende Einstellung.
var fileQuestions =

Falsch: Eine Datei ohne Dateiendung (obwohl diese vorhanden ist). Dies wird aber leider oft so im Windows-Explorer angezeigt.
var fileQuestions = "Fragen";

Richtig:
var fileQuestions = "Fragen.csv";
